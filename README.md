I'm most interested in the gap between a model that works in a notebook and a system that works for a real user — which is why my projects lean toward measurable outcomes: 95% detection accuracy, 40% faster inference, 50% faster decisions. I like AI problems that force me to think like an engineer first (data, pipelines, latency) and a researcher second. Right now that means building across Computer Vision, NLP, and increasingly, agentic AI.
